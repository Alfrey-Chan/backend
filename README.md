## Quick Start

1. Go to [API Documentation](https://backend-zugn.onrender.com/swagger)
2. Click on `/authentication/login` endpoint
3. Click "Try it out"
4. Use demo credentials above
5. Copy the token from response
6. Click "Authorize" button at top
7. Enter: `Bearer {your-token}`
8. Now you can test all protected endpoints!

-   Employee availability submissions
-   Role-based access control (admin vs non admin)
